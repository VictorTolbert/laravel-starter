Create a Makegood Pre-Empt Offer

  1. From the Videa main menu, click Order Management > Order Search. The Order Search page opens.
  2. Find the offer you want to add a makegood pre-empt offer to.
  3. Under the Manage Schedule column for the order, click Manage. The Order Information - Offers screen opens.
  4. Select the line of Traffic LogThe daily broadcast schedule compiled by the broadcaster's traffic department and sent to station automation at least one day before air. data (in the table) that you want to create the makegood against.
  5. In the Details table, click the MG Preempts option.

  6. Do you need to create a freehand spot for the makegood?

    • If yes, follow these instructions:
      1. In the MakegoodAn ad offered by a station in place of a scheduled ad that was pre-empted. Spots Offered section, click Add SpotThe television time bought market-by-market by an advertiser who wants to promote their company or brand.(s). A line opens in the table.
      2. Select the Days where you want the spot.
      3. Select the Start Time of the window where you want the spot to air.
      4. Select the End Time of the window where you want the spot to air.
      5. Enter the Program name of the program where you want to spot to air.
      6. Select the Length of the spot.
      7. In the gate grid, enter the number of spots you want to offer for each date.
      8. Enter the Station RateThe price set for an individual piece of inventory. This price is generally a function of the demand for the inventory combined with its expected audience delivery. for the spot.
      9. Enter the Station RTG or Station IMP for the spot.
      10. Enter any Makegood Comment you want to add.
    • If no, go to the next step.

  7. Do you need to search the station avails to add a spot?

    • If yes, follow these instructions:
      1. In the Station AvailsAvailabilities. Unsold spots available for sellers to sell to buyers or advertisers. Also refers to a seller's submission of program and rating estimates for advertising planning and buying. section, enter the search criteria as described in Search Station Avails Field Descriptions.
      2. Click Search. A listing of available spots populates the table. The table lists all the available avails with the currently approved rates and ratings for the search criteria you selected.
      3. Select the line where you want to make the offer.
      4. Select the Days you do not want to offer.

        The default is to select all days. You can remove days from the offer by unchecking the day in the list.

      5. Enter the number of spots for each date you want to offer in the calendar.
      6. Enter the Spot Rate for the offer.
      7. Enter the Station IMP (impressions) for the offer.
      8. Enter any Makegood Comments you want to add.
      9. In the Makegoods Spots Offered section, click Add Spots. The spot offer moves to the Makegoods Spots Offered section.
    • If no, go to the next step.
  8. At the top of the screen, click Save to save the makegood.
  9. When the makegood is ready, click Transfer to MediaOps. The Videa Media Operations team will send the makegood to the buyer and rep for verification.

    The Transfer to MediaOps button is not available until you save the makegood.
