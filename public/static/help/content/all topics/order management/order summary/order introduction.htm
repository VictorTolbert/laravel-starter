Introduction

The Pending Orders screen gives you a summary and detailed view of the buyer orders.

The Order Summary page is the first thing you or your station manager see after logging in. This screen gives you a high-level view of the Videa-only orders running automatically. You can see your Videa orders from the most recent (the past 24 hours) up to 60 days past. You can also search by an advertiser or by a campaign name.

For each Videa order coming through, the screen shows the flight dates, order date, dollar amount for the order, and share.

SharePercentage of households that are tuned to a specific program at a specific time. is set by the agency when they enter their order parameters and budget.

The ORDER SUMMARY default view shows the orders that have come in through the Videa system within the last 24 hours.

The Videa main menu. Click the links on the screen to open different sections of the Videa platform.

The ORDER SUMMARY search filter allows you to specify the advertiser or campaign you want to see on the screen. See Order Summary Filter for more information.

The ORDER SUMMARY View filter allows you to specify a date range for the information on the screen. See Order Summary Filter for more information.

Export to Excel allows you to export the data on the screen to an Excel spreadsheet. See Export to Excel for more information.

The ORDER SUMMARY table shows a rolled-up view of the Videa orders as defined by the filter criteria. See Order Summary Table for more information.

An expanded order summary line shows further detail about the advertiser's campaign. See Order Summary Expanded for more information.

Click the View Detail link to open the ORDER DETAIL screen for this advertiser's campaign. See Order Summary Detail for more information.
