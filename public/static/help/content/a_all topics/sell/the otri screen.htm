Viewing the OTRI Screen

From the Videa system menu bar, click OTRI. The OTRI screen opens.

Searching in the OTRI Screen

You can search for campaigns that ended between specific dates using the search function at the top of the screen.

  1. Enter the From and To dates between which the campaigns you are looking for ended.
  2. Click Go. The results display in the screen.

About the OTRI Screen

The OTRI screen shows you the progress you are making on meeting the campaign goals to date. The information is gathered from NielsenA research firm that collects media usage data for broadcast TV, cable TV, and interactive advertising. This data is used to make decisions about which content to distribute and where to place advertising so that the target audience is best reached. overnights and applied to the parameters of the campaign, so that you can see at a glance how the campaign is doing.

The graphical view shows you an average of all the Videa system campaigns.

The grid (table) view shows the details of the status of the campaign.

You can toggle between the table view or graphical view of a screen by clicking:

  • For graphical view
  • For table (grid) view

You can set the default view for screens on the PREFERENCES screen.
