Viewing the PRICING / INVENTORY Screen

From the Videa system menu bar, click Reporting > Program Revenue. The Program Revenue screen opens.

Searching in the PRICING / INVENTORY Screen

You can search for a specific time period or for a specific date range from the search parameters at the top of the page.

Searching for a Time Period

  1. Select a time period from the list (for example, week, month, month +2, etc.).
  2. Click Go. The results display in the screen.

Searching for a Specific Date Range

  1. Select a From date using the calendar widget.
  2. Select a To date.
  3. Click Go. The results display in the screen.

About the PRICING / INVENTORY Screen

The Program Revenue screen shows you how your automated Videa accounts are faring relative to your station programming line up. It compares this year's numbers through the Videa automated system to last year's numbers. You can also see the sports or special programs already defined in the Videa system.

You can toggle between the table view or graphical view of a screen by clicking:

  • For graphical view
  • For table (grid) view

You can set the default view for screens on the PREFERENCES screen.
