Using Heroicons

Because Heroicons utilize an external CSS file, they must be implemented using the “inline” approach. You can read more on that here. If you do not include an external style sheet with the necessary classes you will get unstyled results like this:

For your convenience, versions with inline styling are also included in the 'extras' folder. This will make it easier if would like to import the icons into your favorite graphical editing software. These versions will NOT work with external style sheets because the CSS will not be able to override the inline style.

Each icon and their elements have multiple classes to give you more creative freedom when styling them. You can style a specific element of an icon or simply provide an accent color to align them to your brand:

Accent
.heroicon-outline {fill: #303034;} .heroicon-shadow {fill: #303034; opacity:0.4;} .heroicon-component-fill {fill: #FFFFFF;} .heroicon-component-accent {fill: #7ACFC0;}
Full Color
.heroicon-outline {fill: #303034;} .heroicon-home-roof {fill: #7ACFC0;} .heroicon-home-wall {fill: #F2F3D8;} .heroicon-home-frames {fill: #A6BEBA;} .heroicon-home-windows {fill: #DFF6F9;} .heroicon-home-door {fill: #DF7171;} .heroicon-home-parging {fill: #F1F4F4;}

All icon elements have a fill class (.heroicon-component-fill) if you simply want to make them a solid color. If you want to use an accent, make sure you specify the accent class (.heroicon-component-accent) below the fill class in the CSS so it appears properly.

Feel free to check out some of the CSS samples provided in the 'samples' folder. You can test them and experiment with other colors using the demo-lg.html or demo-sm.htmlfiles.

Also, inluded is a blank CSS template (heroicons-template.css) with all of the class for elements if you wish to do full color styling. This supports both large and small icons because they share the same classes. In some cases the large icons have additional classes, these are marked with a block comment.
